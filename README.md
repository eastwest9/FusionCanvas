# FusionCanvas

## 1. 프로젝트소개
### FusionCanvas (2023.09.03 ~ 2023.11.23)
<p text-align='center'> AI(GAN)으로 이미지 생성 및 객체 이미지 배경 제거 후 생성된 이미지랑 이미지 합성하여 프로필 이미지 생성 웹서비스.
<br/>
</p>
